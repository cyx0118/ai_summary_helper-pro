# 🤖 AI 网页总结助手 Pro

一键总结网页内容，侧边栏显示，支持流式输出、多语言、Q&A 问答及导出功能。基于 Tampermonkey (油猴) 运行。
---

## ✨ 核心特性

-   **多模型支持**：原生支持 [DeepSeek](https://platform.deepseek.com/) (深求)、OpenAI (GPT-4o/3.5)、Claude 等兼容接口。
-   **智能总结**：支持简短、中等、详细三种粒度，以及专业、学术、幽默、儿童版等多种语气风格。
-   **深度问答**：基于当前网页内容进行多轮对话（RAG），AI 严格依据网页信息回答，拒绝幻觉。
-   **沉浸体验**：右侧侧边栏悬浮显示，支持拖拽调整宽度，不遮挡原文阅读。
-   **流式输出**：模拟打字机效果，逐字逐句显示结果，减少等待焦虑。
-   **多语言互译**：支持中/英/日/韩/法/德/西/俄等多语言总结与互译。
-   **隐私安全**：配置数据保存在本地浏览器，不收集用户隐私。

## 🚀 快速开始

### 1. 安装依赖
请先安装浏览器扩展 **Tampermonkey** (油猴)：
-   [Chrome Web Store - Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)

### 2. 安装脚本
    1.  下载js文件。
    2.  打开 Tampermonkey 面板，将文件拖入即可。

### 3. 配置 API Key
1.  安装成功后，页面右下角会出现一个 **⚙️ 设置按钮**。
2.  点击进入设置面板。
3.  填入你的 **API Key**（支持 [DeepSeek](https://platform.deepseek.com/)、OpenAI 等）。
4.  选择模型（如 `deepseek-chat`, `gpt-4o`）。
5.  保存并刷新网页。

> **注意**：API 调用通常会产生费用，请确保你的账号有可用额度。

## 🎮 功能使用指南

### 1. 打开侧边栏
-   **方式 A**：点击页面右下角的 **⚙️ 按钮**。
    -   第一次使用需先配置 API Key。
-   **方式 B**：使用 **快捷键**。
    -   默认快捷键：`Ctrl + Q` (可在设置中修改)。
-   **方式 C**：点击浏览器工具栏 Tampermonkey 图标，在菜单中选择 "🤖 总结当前页面"。

### 2. 智能总结 (Summary)
-   **功能**：自动提取网页正文，生成结构化摘要。
-   **操作**：
    1.  侧边栏打开后自动开始总结。
    2.  点击顶部的 **📝 总结** 标签页。
    3.  点击 **🔄 重新总结** 按钮可刷新结果。
-   **自定义**：在设置中调整“总结长度”和“语气风格”。

### 3. 深度问答 (Q&A)
-   **功能**：基于当前网页内容进行提问。
-   **操作**：
    1.  点击顶部的 **💬 问答** 标签页。
    2.  在输入框输入问题（如：“这篇文章的核心观点是什么？”）。
    3.  按 `Enter` 或点击“发送”。
-   **特点**：支持多轮对话，AI 会记住上下文。

### 4. 内容导出
侧边栏底部提供三个按钮：
-   **📄 MD**：导出 Markdown 文件，保留格式。
-   **🖨 PDF**：调用浏览器打印功能，保存为 PDF。
-   **📋 复制**：一键复制总结内容到剪贴板。

### 5. 高级设置
-   **侧边栏宽度**：鼠标拖拽侧边栏最左侧边缘（|◀️| 区域）可调整宽度。
-   **自定义提示词**：在设置面板中修改 `System Prompt`，定制 AI 的行为逻辑。

## ⚙️ 配置说明

| 字段 | 说明 | 默认值/示例 |
| :--- | :--- | :--- |
| **API Key** | 你的 AI 服务商密钥 | `sk-xxxxxxxx` |
| **API 端点** | API 请求地址 | `https://api.deepseek.com/v1/chat/completions` |
| **模型名称** | 选择使用的模型 | `deepseek-chat` |
| **总结语言** | 输出语言 | `中文` |
| **总结长度** | 内容详略程度 | `中等` |
| **快捷键** | 唤起插件的组合键 | `Ctrl + Q` |

## 🛡️ 隐私与安全
-   **数据流向**：网页内容 -> 你的浏览器 -> 你的 API Key -> AI 服务商服务器 -> 返回结果 -> 你的浏览器。
-   **本地存储**：插件仅在浏览器本地存储你的配置（如 API Key、偏好设置），**不会**上传至任何第三方服务器。
-   **免责声明**：AI 生成的内容可能存在错误，请务必核实。
